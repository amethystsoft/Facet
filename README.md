# Amethyst Facet

[TestFlight](https://testflight.apple.com/join/vh1kDYBE)

Amethyst Facet is a minimal Xcode extension doing only one thing: Regex based find & replace.

It highlights matches and capture groups on the left and gives you a preview of the changes on the right.
